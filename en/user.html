User

A Sertifi User is the most basic role level in the Sertifi portal. A user is typically an individual that signature documents are sent to for signature or payment. Anyone who receives a signature document or payment via Sertifi, can create their own user account, so they can quickly sign and submit a payment. Sertifi Users are can update their personal information and only view files they've signed.
